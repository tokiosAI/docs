Tokios exposes standard OpenAI and Anthropic API surfaces, so any client that lets you set a custom base URL works. You change two things: the **base URL** and your **API key** (`sk-tok-…`). The `"model"` value is the public name you registered (for example, `gemma-tunnel`).

## Which base URL?

Pick the base URL by the API shape your client speaks:

| Client speaks… | Base URL | Examples |
|---|---|---|
| OpenAI style | `https://api.tokios.com/v1` | OpenAI Codex, OpenAI SDKs, Aider, Cline, Continue, Open WebUI, AnythingLLM, LangChain |
| Anthropic style | `https://api.tokios.com` (no `/v1`) | Claude Code, Anthropic SDKs |

## Per-client setup

| Client | Base URL setting | API key setting |
|---|---|---|
| Claude Code | `ANTHROPIC_BASE_URL` = `https://api.tokios.com` | `ANTHROPIC_API_KEY` |
| OpenAI Codex | `base_url` in `~/.codex/config.toml` — see [OpenAI Codex](/guides/openai-codex) | `OPENAI_API_KEY` |
| Aider | `--openai-api-base https://api.tokios.com/v1` | `OPENAI_API_KEY` |
| Cline / Continue | OpenAI-compatible base URL field | API key field |
| Open WebUI / AnythingLLM | OpenAI-compatible base URL field | API key field |
| OpenAI SDK (Python / Node) | `base_url` / `baseURL` param | `api_key` / `apiKey` param |
| LangChain (Python) | `openai_api_base` | `openai_api_key` |

<Tip>
  Clients that read `OPENAI_BASE_URL` or `ANTHROPIC_BASE_URL` pick Tokios up automatically once you export those variables in your shell.
</Tip>

## Examples

### OpenAI Node.js SDK

```javascript openai_node.js
import OpenAI from 'openai';

const client = new OpenAI({
  baseURL: 'https://api.tokios.com/v1',
  apiKey: 'sk-tok-YOUR_KEY',
});

const response = await client.chat.completions.create({
  model: 'gemma-tunnel',
  messages: [{ role: 'user', content: 'Hello!' }],
});
console.log(response.choices[0].message.content);
```

### LangChain (Python)

```python langchain_tokios.py
from langchain_openai import ChatOpenAI

llm = ChatOpenAI(
    model="gemma-tunnel",
    openai_api_base="https://api.tokios.com/v1",
    openai_api_key="sk-tok-YOUR_KEY",
)
print(llm.invoke("Summarize this code: print('hello')").content)
```
